Congratulations! You have recently been hired as a data jedi for the Galactic News Hub, the premiere news organization for the Star Wars Universe. Your first task: apply the tidyverse skills that you learned this week to a dataset of Star Wars characters.

1. Previewing our data

For this exercise, we will using a dataset called starwars. This dataset comes with tidyverse, so if you have loaded tidyverse into your R environment using library(), the starwars dataset should be there! As you can see below, starwars contains some information about Star Wars characters, like their name and height.

library(tidyverse)
starwars

First, let’s learn about the starwars dataset. The first thing we’ll want to know is how many characters are in the starwars dataset. What function would you use to glimpse the data to find that answer?

Hint: Check out the glimpse() function, which we talk about in Chapter 3

glimpse(starwars)

2. Focusing on variables

Now that we know how to glimpse into our dataset, let’s wok on selecting specific variables of interest. Let’s say your newspaper has asked you to write a story about height differences across different species. What function would you use to select only the following variables in starwars: name, height, homeworld, and species?

Hint: I recommend checking out the select() function, which we talk about in Chapter 3

starwars %>%
  select(
    name,
    height,
    homeworld,
    species
  )

3. Arranging our Data

Now that we have glimpsed our data and selected our variables in the starwars, let’s arrange them using by height. What function would we use to arrange our data so that the tallest person is listed at the top? (Hint: since tidyverse arranges numbers from the smallest to the greatest number, you may need another function so the information is in descending order.)

Hint: I recommend checking out the arrange() function, which we talk about in Chapter 3

starwars %>%
  arrange(height %>% desc())
